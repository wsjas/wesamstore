وسام للإلكترونيات - تحسينات الإصدار الثاني

افتح لوحة التحكم من:
/admin.html

الإضافات الجديدة:
1) لوحة Dashboard محسنة.
2) دخول مبدئي عبر Firebase Auth مع خيار دخول مؤقت للتجربة.
3) إدارة المنتجات: إضافة، تعديل، حذف، تصدير CSV.
4) طلبات البيع: إضافة يدوية، تغيير حالة، طباعة، دفعات، حذف.
5) طلبات الصيانة: إضافة وتعديل، إسناد فني، دفعات، طباعة، حذف.
6) العملاء: تجميع تلقائي من الطلبات والصيانة والمشتركين، وسجل عميل، واتساب مباشر.
7) المحاسبة: بيع + صيانة + إيرادات يدوية + مصروفات + صافي + تصدير CSV + طباعة تقرير.
8) صفحة مصروفات مستقلة.
9) صفحة دفعات وذمم لمعرفة المبالغ المتبقية على العملاء.
10) كوبونات وعروض.
11) إدارة الفنيين.
12) إشعارات داخلية.
13) نسخة احتياطية JSON من زر داخل اللوحة.
14) تهيئة تلقائية للمجموعات الأساسية في Firebase.

مهم:
- انشر firestore.rules أثناء التجربة حتى تعمل كل وظائف الحفظ.
- بعد التأكد، استخدم firestore.secure.rules لحماية لوحة التحكم بشرط إعداد Firebase Auth والمدير.
- إذا ظهرت صفحة الدخول، استخدم دخول مؤقت للتجربة أو فعّل مستخدم مدير في Firebase Authentication.
