# تحسينات وسام للإلكترونيات - النسخة الخامسة V5

هذه النسخة مبنية فوق V4 وتضيف أدوات تشغيل يومية للوحة التحكم والمحاسبة والصيانة.

## الصفحات الجديدة

1. `admin-operations.html`
   - مركز متابعة كل طلبات البيع والصيانة.
   - يكشف الطلبات المفتوحة والمتأخرة أكثر من 3 أيام.
   - تحديث الحالة مباشرة وحفظها في Firebase.
   - تصدير الطلبات المتأخرة CSV.

2. `admin-debts.html`
   - مركز الذمم والدفعات.
   - يجمع المتبقي من طلبات البيع والصيانة.
   - تسجيل دفعة وحفظها في `payments`.
   - إضافة حركة دخل تلقائية في `accounting_transactions`.

3. `admin-smart-reports.html`
   - تقارير مالية حسب فترة.
   - يجمع البيع والصيانة والدفعات والمصروفات والحركات اليدوية.
   - طباعة وتصدير CSV.

4. `admin-alerts-center.html`
   - تنبيهات المنتجات قليلة الكمية.
   - تنبيهات الطلبات المتأخرة.
   - تنبيهات الذمم غير المدفوعة.
   - إنشاء إشعارات داخل `notifications`.

5. `admin-field-tech.html`
   - واجهة خفيفة للفني على الهاتف.
   - عرض طلبات الصيانة المفتوحة.
   - تحديث حالة الصيانة وملاحظة الفني.
   - إضافة طلب صيانة سريع.

6. `admin-health-check.html`
   - فحص Firebase متقدم.
   - اختبار كتابة/قراءة/حذف.
   - إنشاء المجموعات الأساسية.
   - تصدير تقرير التشخيص.

## مجموعات Firebase الجديدة

- `work_notes`
- `followups`
- `diagnostics`

## مهم بعد الرفع

انسخ محتوى `firestore.rules` إلى Firebase Console > Firestore Database > Rules ثم اضغط Publish.

بعد تجهيز حساب المدير يمكنك استخدام `firestore.secure.rules` بدل القواعد المفتوحة.

## روابط مهمة

- لوحة التحكم: `/admin.html`
- مركز العمليات: `/admin-operations.html`
- الذمم والدفعات: `/admin-debts.html`
- التقارير الذكية: `/admin-smart-reports.html`
- مركز التنبيهات: `/admin-alerts-center.html`
- واجهة الفني: `/admin-field-tech.html`
- فحص Firebase: `/admin-health-check.html`
