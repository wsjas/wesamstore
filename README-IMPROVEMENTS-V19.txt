تحسينات وسام للإلكترونيات - V19
================================

تمت إضافة تحسينات جديدة على الصفحة الرئيسية ولوحة التحكم بدون حذف أو كسر ملفات V18.

أولًا: تحسينات الصفحة الرئيسية
------------------------------
1) ملف جديد: wesam-v19-home-enhancements.css
2) ملف جديد: wesam-v19-home-enhancements.js
3) تم ربط الملفين داخل index.html.

ما الذي أضيف للصفحة؟
- مساعد ذكي لاختيار نوع العطل والخدمة المناسبة.
- أزرار مباشرة لتسجيل طلب صيانة، واتساب، وتتبع الطلب.
- شريط ثقة يوضح رقم الطلب، الضمان، الخدمة المحلية، والمتابعة.
- شريط سريع للموبايل أسفل الصفحة: صيانة، واتساب، تتبع، الخريطة.
- مؤشر اتصال صغير يوضح هل المتصفح متصل أم غير متصل.
- قراءة بيانات واتساب والخريطة من settings/site عند توفر Firebase، مع قيم احتياطية آمنة.

ثانيًا: تحسينات لوحة التحكم
----------------------------
1) ملف جديد: admin-v19-enhancements.css
2) ملف جديد: admin-v19-enhancements.js
3) صفحة جديدة: admin-v19-center.html
4) تم ربط ملفات V19 داخل admin.html.

ما الذي أضيف للوحة؟
- مركز أوامر سريع يفتح بالزر الجديد أو الاختصار Ctrl + K.
- بحث سريع داخل الجدول المفتوح حاليًا.
- بطاقة V19 في الرئيسية فيها اختصارات: طلب صيانة، منتج جديد، مصروف، فحص Firebase، فحص الأزرار.
- فحص سريع لأعداد المنتجات والطلبات والعملاء من أرقام القائمة الجانبية.
- ملاحظات يومية محلية تحفظ على جهاز المدير فقط.
- أزرار أعلى اللوحة لفتح مركز V19 وفحص الروابط والحجز العام.

ثالثًا: صفحة admin-v19-center.html
-----------------------------------
صفحة تشغيل مستقلة تساعدك على:
- فحص أعداد المنتجات وطلبات البيع والصيانة والرسائل من Firebase.
- إنشاء تذكير إداري داخل مجموعة notifications.
- تجهيز رسالة واتساب مباشرة للعميل.
- حفظ قائمة فحص يومية محلية على جهازك.
- نسخ تقرير تشخيص سريع عند حدوث مشكلة.

ملاحظات مهمة
------------
- لم أضع أي مفاتيح جديدة أو كلمات مرور داخل الملفات.
- لم أغيّر منطق تسجيل الدخول الأساسي.
- لم أفتح صلاحيات خطرة في قواعد Firestore.
- صفحة V19 الجديدة تعتمد على الصلاحيات الموجودة لديك، فإذا ظهرت رسالة صلاحيات فهذا يحتاج تعديل قواعد Firestore للمدير وليس خطأ في الصفحة.

طريقة الرفع
-----------
ارفع الملفات كما هي إلى مجلد الاستضافة ثم نفّذ:
firebase deploy --only hosting

أو إذا كنت ترفع إلى GitHub Pages انسخ الملفات الجديدة مع الملفات المعدلة:
- index.html
- admin.html
- wesam-v19-home-enhancements.css
- wesam-v19-home-enhancements.js
- admin-v19-enhancements.css
- admin-v19-enhancements.js
- admin-v19-center.html
