رفع منتجات من ملف JSON

يمكنك رفع ملف JSON يحتوي على قائمة بالمنتجات لإضافتها دفعة واحدة إلى المتجر. سيتم إضافة المنتجات تلقائياً إلى قاعدة البيانات مع تصنيفاتها وأوصافها.

صيغة ملف JSON المطلوبة:

[
  {
    "name": "اسم المنتج",
    "category": "التصنيف",
    "sell": 25.00,
    "stock": 10,
    "description": "وصف المنتج والمواصفات",
    "image": "https://example.com/image.jpg",
    "oldPrice": 30.00,
    "discountPercent": 15,
    "onSale": true,
    "featured": false,
    "deliveryFee": 2,
    "videoUrl": "",
    "softwareUrl": "",
    "additionalImages": [
      "https://example.com/img2.jpg"
    ]
  }
]

شرح الحقول:

الحقلمطلوبالوصف
name✅ نعماسم المنتج
category✅ نعمتصنيف المنتج (مثال: سماعات، ريموتات، شواحن)
sell✅ نعمسعر البيع بالدينار
stock✅ نعمالكمية المتوفرة في المخزون
description❌ اختياريوصف تفصيلي للمنتج ومواصفاته
image✅ نعمرابط الصورة الرئيسية (URL مباشر)
oldPrice❌ اختياريالسعر قبل الخصم
discountPercent❌ اختيارينسبة الخصم (0-99)
onSale❌ اختياريعرض ضمن العروض والخصومات (true/false)
featured❌ اختياريعرض ضمن المنتجات المختارة (true/false)
deliveryFee❌ اختياريرسوم التوصيل (افتراضي: 2 دينار)
videoUrl❌ اختياريرابط فيديو المنتج (يوتيوب)
softwareUrl❌ اختياريرابط سوفت وير المنتج
additionalImages❌ اختياريقائمة روابط صور إضافية

قسم الصيانة - قدّم طلبك خلال دقيقة

صيانة أجهزة TV / LCD / LED والإلكترونيات في عمّان. ارسل تفاصيل العطل مع الموقع (اختياري) وسنقوم بالتواصل معك وتجهيز الطلب بسرعة.

فحص وتشخيص توصيل من وإلى المنزل تواصل سريع خبرة وإتقان

ماذا نحتاج منك؟

  • الاسم + رقم الهاتف
  • نوع الجهاز + الحجم
  • وصف العطل
  • صورة (اختياري) + رابطها
  • موقعك (اختياري)

بعد إرسال الطلب، سيتم فتح واتساب تلقائياً مع رسالة جاهزة (ويوجد خيار إرسال ايميل أيضاً).

تسجيل طلب صيانة

بعد رفع الصورة في PostImages انسخ رابط (Hotlink for websites) والصقه هنا.
يمكنك أيضاً وضع رابط خرائط أو كتابة عنوان مختصر.
سيتم حفظ الطلب في قاعدة البيانات ليسهل عليك متابعته من لوحة التحكم.
